Installation

DVS

STEP 1
  • Create and download a full backup of your site.
  • Not that you particularly need to, its just a good habit to make a full backup anytime you make changes to your site!

STEP 2
  • Create a new directories named logo inside of the /file/dvs directory and CHMOD it to 0777.
  • The full path should be /file/dvs/logo

STEP 3
  • Create a new directories named branding inside of the /file/dvs directory and CHMOD it to 0777.
  • The full path should be /file/dvs/branding

STEP 4
  • Create a new directory named preroll inside of the /file/dvs directory and CHMOD it to 0777.
  • The full path should be /file/dvs/preroll

STEP 5
  • From the AdminCP, remove the Platform product.

STEP 6
  • Using an FTP program, remove the platform module directory and all of it's contents. This will be at /module/platform

STEP 7
  • Create a new directory named background inside of the /file/dvs directory and CHMOD it to 0777.
  • The full path should be /file/dvs/background

STEP 8
  • Upload the contents of the /upload folder to your root phpfox directory directory. Choose to overwrite any existing files.
  • Usually, this directory will be /public_html, /www, or /htdocs.

STEP 9
  • Go to the "Import Products" section of your Phpfox Admin Panel.
  • The section can be found by going to the Main Menu "Extensions" -> "Products" -> "Import Products".
  • Install the product by clicking the "Install" link under the "Action" column.

STEP 10
  • Add your Google Maps API key in the DVS Settings

STEP 11
  • If upgrading to DVS 5.x from DVS 4.x or earlier, complete steps 12 through 15.

STEP 12
  • Delete the /module/dvs and /module/idrive directories.
  • Some files have been deleted. Simply uploading the new DVS and iDrive products will not remove them.

STEP 13
  • Re-upload the DVS and iDrive product upload directories if needed. Both products may now be safely upgraded in the AdminCP.

STEP 16
  • A phpFox 3.x core file will need to be modified.
  • In the file /include/library/phpfox/template/template.class.php on line 1805 add:

  • (($sPlugin = Phpfox_Plugin::get('template_getheader_meta_custom')) ? eval($sPlugin) : false);

  • after:

  •     if ($sMeta == 'og:description')
        {
            $sMetaValue = $oPhpfoxParseOutput->shorten($oPhpfoxParseOutput->clean($sMetaValue), Phpfox::getParam('core.meta_description_limit'));
        }

STEP 17
  • Enable the TinyMCE Editor for DVS Welcome Greetings.
  • Select "Setting" -> "Manage Settings" -> "TinyMCE".
  • In the "Add a New Value..." box, add the following:

  •     dvs.settings|#welcome

  • Click "Add" and then "Submit"

STEP 18
  • Clear the cache.
  • Select "Tools" -> "Maintenance" -> "Cache Manager".
  • Click the "Clear All" button.

  • The new installation is now ready to use!
