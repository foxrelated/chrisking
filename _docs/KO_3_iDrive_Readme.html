Installation

iDrive

STEP 1
  • Create and download a full backup of your site.
  • Not that you particularly need to, its just a good habit to make a full backup anytime you make changes to your site!

STEP 2
  • Create a new directory named logo inside of the /file/idrive directory and CHMOD it to 0777.
  • The full path should be /file/idrive/logo

STEP 3
  • Move the contents of the /file/idrive directory in to the newly created logo directory.
  • The quickest way to do this may be to download the contents (likely only a folder named 2012), delete from the server, and upload in the logo directory.

STEP 4
  • Create a new directory named preroll inside of the /file/idrive directory and CHMOD it to 0777.
  • The full path should be /file/idrive/preroll

STEP 5
  • Remove the previous version of the product.
  • The section can be found by going to the Main Menu "Extensions" -> "Product" -> "Mange Products".
  • Uninstall the product by selecting the small arrow next to the product name and clicking "Delete".
  • Some of the existing phrases have changed (settings and usergroup settings) and new settings were added.

STEP 6
  • Upload the contents of the /upload folder to your root phpfox directory directory. Choose to overwrite any existing files.
  • Usually, this directory will be /public_html, /www, or /htdocs.

STEP 7
  • Go to the "Import Products" section of your Phpfox Admin Panel.
  • The section can be found by going to the Main Menu "Extensions" -> "Products" -> "Import Products".
  • Install the product by clicking the "Install" link under the "Action" column.

STEP 8
  • Clear the cache.
  • Select "Tools" -> "Maintenance" -> "Cache Manager".
  • Click the "Clear All" button.

  • The new installation is now ready to use!
